HTML es un lenguaje utilizado para dar estructura a paginas Web. HTML es la sigla de Hyper Text Markup Language que en español significa lenguaje de marcas de hipertexto (un hipertexto es un texto que contiene enlaces).

Consta de etiquetas para la definición de contenido como texto, imagen, enlace,etc.

La función de un navegador es transformar el código HTML en una página Web utilizable. Para crear una página Web podemos utilizar un editor de texto en nuestro PC, por ejemplo gedit en Ubuntu o notepad en Windows

1. Etiquetas

Son como marcas que utilizamos para delimitar cada uno de los elementos de una página. Algunos ejemplos de elementos son: titulos, párrafos, imágenes, tablas, listas, enlaces. Un elemento debe estar entre una etiqueta de inicio y una de cierre, aunque hay algunas etiquetas autocerradas. Cada etiqueta de inicio se distingue por comenzar con el signo menor que (<) y terminar con mayor que (>), y la respectiva etiqueta de cierre es igual excepto que comienza con menor que y signo de división (</>).

2 Algunas etiquetas básicas

<!DOCTYPE html> Con esta etiqueta se empieza una página web, debe ser la primera línea y esta le indica al navegador que vamos a utilizar Html.
<html></html> Estas etiquetas van al comienzo y al final de de la página. La página siempre consta de dos partes: la cabecera y el cuerpo.
<head></head> Es la cabecera de la página. En la cabecera está el título.
<title></title> En el titulo se puede escribir el nombre de la página, sólo se puede poner dentro de la cabecera de la página.
<body></body> Es el cuerpo de la página. El cuerpo es todo lo que vemos en la ventana del navegador cuando abrimos la página con este.
<h1></h1>Títulos y encabezados de nuestros párrafos, hay desde h1 hasta h6 el uno es el más grande y el seis es el más pequeño.
<p></p>Párrafo
<img></img> Se utiliza para agregar una imagén.
<a></a> Enlaces o hipervínculos, quiere decir que al pulsar con el ratón sobre el contenido (que podría ser un texto, una imagen) te lleva a otra página o a otra parte de la misma página.
<strong></strong> Sirve para resaltar en negrilla el texto contenido.
<span></span> Sirve para delimitar texto, por ejemplo para cambiar sus propiedades de estilo.
<em></em> Sirve para resaltar el texto contenido.

3. Elementos anidados

Cuando hay elementos dentro de otro elemento (es decir anidados), las etiquetas se organizan por parejas de incio y de cierre, al primer elemento que se abre le debe corresponder el último que se cierra. Es impotante la indentacion para poder entender mejor cuando hacemos anidados y para poder leer mejor la información.

        <title>

4. Atributos

Es una característica de una etiqueta, se utiliza para personalizar más el contenido de la etiqueta. Consta de un nombre y un valor. Los atributos los podemos agregar a un párrafo, un título, una tabla, etc. Cuando agregamos un atributo a una etiqueta, este se agrega en la etiqueta de inicio y la etiqueta de cierre queda igual.

Los atributos se agregan a la derecha del nombre de la etiqueta separándolos con un espacio, comenzando por el nombre del atributo el signo igual y a continuación el valor entre comillas (" ").

Por ejemplo al agregar el atributo con nombre style y valor color:red a la etiqueta <th> quedará asi: <th style="color:red">. El atributo style permite personalizar con CSS un elemento cualquiera, con este podemos cambiar:

<h1 style="color:red"></h1> Color de fuente, los colores se escriben en inglés.
<p style="font-background-color: blue"></p> El color del fondo
<p style="font-family:Garamond"></p> Tipo de fuente
<p style="font-size:12px"></px> Tamaño de fuente, px se refiere a píxeles.

Otros atributos son específicos para algunas etiquetas, por ejemplo:

<img> Se utiliza para agregar una imagen. Como no tiene contenido indica donde está la imagen, usando el atributo src.

Ejemplo: <img src="http://bit.ly/1eQMfPi"></img>

<a> Sirve para crear vínculos. Estos vínculos pueden ser palabras, párrafos, imágenes, etc. Estos vínculos pueden ser a otra página del mismo sitio,(por ejemplo Ir a capítulo 2), podemos abrir un vínculo en otra pestaña del navegador (Ir a Amen-Amen), también a un correo (blanca@pasosdeJesus.org) o simplemente hacer click y cambiar de sitio. Usamos el atributo href.

Ejemplo <a href="http://www.amen-amen.net/RV1960/"> </a> Asi se verá una imagen con enlace, y al pulsar sobre ella, podemos ir a otra pagina web o a otra parte de la página.

Cuando van varios atributos en una sola etiqueta se separan con espacio. El valor completo de un atributo debe ir entre comillas.

En el caso del atributo style, que permite cambiar propiedades CSS, se pueden cambiar varias propiedades simultáneamente separándolas con punto y coma (;). Cada propiedad se modifica con el nombre de la propiedad, el signo dos puntos (:) y a continuación el valor. Si en a la etiqueta <th style="color:red"> quisiéramos también cambiar el tipo de fuente a Garamond, quedaría: <th style=”color:red ; font-family:Garamond;”>.

5. Anclas

Es una referencia dentro de la página para poder posteriormente disponer de un enlace a dicha referencia.

La sintaxis para definir un enlace a un ancla es: <a href="#nombre del ancla"> Nombre del Ancla </a>
Al pulsar sobre ese vínculo irá al sitio donde se defina el ancla, dejandola al comienzo de la página.

Ahora la sintaxis para el ancla es similar pero usando el atributo name y sin el símbolo numeral en el nombre del ancla:
<a name="nombre del ancla"></a>

Lo invitamos a examinar la fuente de esta página (con chrome botón derecho y elegir "Ver código fuente de la página"), en particular la tabla de contenido que está al comienzo con enlaces a diversas anclas, así como las anclas al inicio de cada sección.

Para ir a un ancla en otra página (digamos pagina2.html) la sintaxis es: <a href="pagina2.html#comentarios">Comentarios</a>
