Lecciones básicas de HTML

HTML es un lenguaje utilizado para dar estructura a paginas Web. HTML del inglés Hyper Text Markup Language que en español significa lenguaje de marcas de hipertexto (un hipertexto es un texto que contiene enlaces).

Consta de etiquitas para la definipreviewción de contenido como texto, imagén, enlace,etc.

La función de un navegador es transformar el código HTML en una página Web utilizable. Para crear una página Web podemos utilizar un editor de texto en nuestro PC, por ejemplo gedit en Ubuntu o notepad en Windows

    1. Etiquetas

    Son como marcas que utilizamos para delimitar cada uno de los elementos de una página. Algunos ejemplos de elementos son: titulos, palabras, parrafos, imagenes y enlaces. Un elemento debe estar entre una etiqueta de inicio y una de cierre, aunque unas pocas etiquetas (como la de imagenes) pueden usarse sin etiqueta de cierre y no tienen contenido. Cada etiqueta de inicio se distingue por comenzar con el signo menor que (<) y terminar con mayor que (>), y la respectiva etiqueta de cierre es igual excepto que comienza con menor que y signo de división (</>).

    2. Algunas etiquetas básicas

    <!DOCTYPE html>Con esta etiqueta se empieza una página web, debe ser la primera linea y esta le indica al navegador que vamos a utilizar Html.
    <html></html> Estas etiquetas van al comienzo y al final de de la página. La página siempre consta de dos partes: la cabecera y el cuerpo.
    <head></head>Es la cabecera de la pagina. En la cabecera está el título.
    <title></title>En el titulo se puede escribir el nombre de la página, sólo se puede poner dentro de la cabecera de la página.
    <body></body>Es el cuerpo de la página. El cuerpo es todo lo que vemos en la ventana del navegador cuando abrimos la página con este.
    <h1></h1>Títulos y encabezados de nuestros párrafos, hay desde h1 hasta h6 el uno es el más grande y el seis es el más pequeño.
    <p></p>Párrafo
    <img></img> Se utiliza para agregar una imagén.
    <a></a> Enlaces o hipervínculos, quiere decir que al pulsar con el ratón sobre el contenido (que podría ser un texto, una imagen) te lleva a otra página o a otra parte de la misma página.
    <strong></strong> Sirve para resaltar el texto contenido.
    <span></span>Sirve para delimitar texto, por ejemplo para cambiar sus propiedades de estilo.
    <em></em> Sirve para resaltar el texto contenido.

    2. Atributos

    Es una característica de una etiqueta, se utiliza para personalizar más el contenido de la etiqueta. Consta de un nombre y un valor. Los atributos los podemos agregar a un párrafo, un título, una tabla, etc. Cuando agregamos un atributo a una etiqueta, este se agrega en la etiqueta de inicio y la etiqueta de cierre queda igual. Los atributos se agregan a la derecha del nombre de la etiqueta separándolos con un espacio, comenzando por el nombre del atributo el signo igual y a continuación el valor entre comillas (" ").

    Por ejemplo al agregar el atributo con nombre style y valor color:red a la etiqueta <th> quedará asi: <th style=”color:red”>. El atributo style permite personalizar con CSS un elemento cualquiera, con este podemos cambiar:

    <h1 style=”color:red”></h1>Color de fuente, los colores se escriben en inglés.
    <p style=”font-background-color: blue”></p>El color del fondo
    <p style=”font-family:Garamond”></p>Tipo de fuente
    <p style=”font-size:12px”></px>

    Otros atributos son específicos para algunas etiquetas, por ejemplo:

    <img> Se utiliza para agregar una imagen. Como no tiene contenido indica donde está la imagen, usando el atributo src.

    Ejemplo: <img src=”https://encrypted-tbn1.gstatic.com/images?q=tbn:ANd9GcTwbYh5M6SoF4IGNhCXE6Fiai pwptAT9SzSpIsXl94ZvuIcpbYM4ltx7SPu”></img>

    <a> Sirve para crear vínculos. Estos vínculos pueden ser palabras, párrafos, imágenes, etc. Usamos el atributo href. Ejemplo <a href=”http://www.amen-amen.net/RV1960/”> </a> Asi se vera una imagen con imagen, y al pulsar sobre ella, podemos ir a otra pagina web o a otra parte de la página.

    Cuando van varios atributos en una sola etiqueta se separan con espacio. El valor completo de un atributo debe ir entre comillas.

    En el caso del atributo style, que permite cambiar propiedades CSS, se pueden cambiar varias propiedades simultáneamente separándolas con punto y coma (;). Cada propiedad se modifica con el nombre de la propiedad, el signo dos puntos (:) y a continuación el valor. Si en a la etiqueta <th style=”color:red”> quisiéramos también cambiar el tipo de fuente a Garamond, quedaría: <th style=”color:red ; font-family:Garamond;”>.

    3. Elementos anidados

    Cuando hay elementos dentro de otro elemento (es decir anidados), las etiquetas se organizan por parejas de incio y de cierre, al primer elemento que se abre le debe corresponder el último que se cierra. Es impotante para poder entender mejor cuando hacemos anidados y para poder leer mejor la información.

        <head>
            <title>
            </title>
        </head>
    <html>

    Anclas

    Es una referencia dentro de la página para poder posteriormente disponer de un hipervínculo a dicha referencia.

    La sintaxis para definir un ancla es: <a href="#nombre del ancla"> Nombre del Ancla </a>
    y al hacer click sobre ella nos llevara al ancla allí nombrada
    Y en la página se ve así: Comentarios

    Ahora la sintaxis para ir a un ancla desde un hipervínculo y debe estar justo antes de la etiqueta. Al hacer click sobre el ancla lo lleva etiqueta y esta no se ve en la página pero si la etiqueta es la siguiente:
    <a name="nombre del ancla">Nombre del ancla</a>

    Para ir al ancla desde otra página la sintaxis es: <a href="pagina2.html#comentarios">Comentarios</a>
